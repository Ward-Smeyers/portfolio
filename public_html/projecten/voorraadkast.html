De voorraadkast

De voorraadkast is een project dat is ontworpen om huishoudbeheer te vereenvoudigen door gebruik te maken van moderne technologieën. Het biedt een geïntegreerde oplossing voor het beheren van voorraad, het scannen van barcodes, en het automatisch verzenden van boodschappenlijsten via WhatsApp. Dit project combineert gebruiksvriendelijke software met krachtige infrastructuur om een efficiënte en toegankelijke ervaring te bieden.

Verbeterde skills in dit project

  • In een team werken
  • Effectief communiceren binnen het team
  • Mijn eigen kennis in woorden omzetten
  • Projectmanagement

Infrastructuur

De omgeving van de voorraadkast bevindt zich in het datacenter van Thomas More. Op de ESXi-server draait een virtuele machine waarop de benodigde containers draaien. Tot slot werdt er via Cloudflare een tunnel gemaakt naar het datacenter waardoor de applicatie zonder een VPN online beschikbaar was.

Virtuele Machine

Op een Ubuntu virtuele machine zullen er verschillende containers draaien die samen een applicatie vormen waar de volgende zaken in staan:

  • WhatsApp-connector: een op maat gemaakte Python-applicatie
  • Grocy
  • Barcode Buddy
  • Cloudflared

Whatsappconnector

De klant/docent stelde als vereiste dat de boodschappenlijst automatisch moest worden verzonden, bij voorkeur via WhatsApp. Twilio is een cloud communicatieplatform waarmee ontwikkelaars eenvoudig SMS, spraakoproepen, WhatsApp-berichten en videochats kunnen integreren in hun applicaties via API’s.

De WhatsApp-connector is een door ons team ontwikkelde Python-applicatie die draait in een container op de virtuele machine en communiceert met de Twilio API. Zo kan de applicatie boodschappenlijsten automatisch verzenden naar de opgegeven WhatsApp-nummers. Deze WhatsApp-berichten kunnen naar één of meerdere nummers worden verzonden en op instelbare tijdstippen worden gepland met Python Schedule.

Python Schedule

Python Schedule is een python lib en diend voor het periodiek uitvoeren van taken. De boodschappenlijst moet op een vast tijdstip worden verstuurd naar de ingestelde WhatsApp-nummers.

Cloudflare

Cloudflare fungeert als een reverse proxy, die het verkeer tunneled naar de applicatie Grocy en Barcode Buddy in het datacenter. Dit houdt in dat Cloudflare als een tussenstation werkt tussen de eindgebruiker en de daadwerkelijke server (de containers op de virtuele machine). Bijkomend beschermt het de applicatie door verkeer via hun netwerk te leiden, waarbij caching, DDoS-bescherming en andere optimalisaties worden toegepast. Hiervoor draait een aparte container op de virtuele machine die Cloudflared draait.
